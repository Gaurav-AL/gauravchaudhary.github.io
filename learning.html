Arrays, Strings, Sliding Window, Two Pointers — must be fluent in O(n) solutions
Trees (BFS/DFS/LCA), Graphs (Dijkstra, Union-Find, Topological Sort)
Dynamic Programming — 1D, 2D, knapsack, string DP patterns
Heaps, Priority Queues, Monotonic Stacks
Binary Search on answer — critical for mid-level rounds
